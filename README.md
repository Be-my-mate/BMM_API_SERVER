## 컨벤션
### 커밋 컨벤션 
|   Commit type              | title                                         | Situation |
|:---------------------------|:----------------------------------------------|:----|
| Initial commit             | init                               | 작업 세팅 커밋 (패키지 설치 등) |
| feat                | feat                       | 새로운 기능, UI 등 | 
| bug                     | bug                                 |  버그 수정/리포트  |
| Documentation              | docs                             | 문서화 |
| refactor             | refac                           | 구조 변경, 리팩토링 |
| remove or edit        | edit                              |  파일 삭제 및 수정 |



### 브랜치 컨벤션

- 기능 추가 : feat/issue
- 버그 수정 : fix/layout
- 긴급 수정 : hotfix
